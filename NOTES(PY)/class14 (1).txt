Control Statements in python:
========================
Python supports control statements.The main objective of control statement is write the business or source code of our application by reducing the code.Using control statement we 
can our application as per our application requirement.


Following types of control statements are in python


->Decision making statements or conditional statements
->Iterative statements
->Transfer statements



->Decision making statements or conditional statements:
===============================================
Following are the decision making or conditional statements in python

->if statement
->nested if statement
->if-else statement
->if-elif-else statement
